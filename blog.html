Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is the structure of a webpage where each part of the webpages are like a tree branch or leaf. Like headings,paragraphs, images, and links.

How do you select an element from the DOM?

We select five ways an element from the DOM :

  • 1.getElementById()
  • 2.getElementsByClassName()
  • 3.getElementsByTagName()
  • 4.querySelector()
  • 5.querySelectorAll()
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is way where you can add a single event listener to a parent element instead of adding each child element individually. It is helpful because it saves time, resources, clean and simple organized to code.

How do you manipulate an element's attributes and styles using the DOM?

We can use getAttribute() to retrieve an attribute's value, to change or add setAttribute() and to delete removeAttribute(). For style we can use style property in the element.
